传值 or 传引用

Java 中都是传值，即使是对象也只是传对象地址值的副本。

抽象类 vs 接口

相同点

  • 都不能被实例化

区别

  • 抽象类可以有构造器，而接口不能有构造器
  • 抽象类只能单继承，接口可以多继承
  • 接口的变量都是 final 的，抽象类的变量可以不是 final
